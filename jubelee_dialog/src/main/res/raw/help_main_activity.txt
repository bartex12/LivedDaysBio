
O ПРИЛОЖЕНИИ
Приложение предназначено для расчёта количества прожитых дней и построения графика биоритмов(для одной личности), количества совместно прожитых дней (для двух личностей) на основе даты рождения.

ВВОД ПЕРСОНАЛЬНЫХ ДАННЫХ
Перед началом работы необходимо вввести персональные данные (имя и дата рождения) хотя бы одной личности.
Персональные данные личности  с экрана главного меню можно ввести при нажатии на круглую кнопку, расположенную внизу экрана справа.

ГЛАВНОЕ МЕНЮ
содержит список программ, доступных в приложении

При щелчке на пункте ЛИЧНЫЕ ДАТЫ вы сначала попадаете на экран выбора личности из списка персональных данных. Щелчок на строке списка открывает экран
с количеством прожитых дней, где вы можете убедиться, как бысто бежит время, а также рассчитать дату наступления какого-либо события или посмотреть таблицу "круглых дат"

При щелчке на пункте СОВМЕСТНЫЕ ДАТЫ вы попадаете на экран выбора личностей, для которых хотите посмотреть количество совместно прожитых дней. Отметив галочками
выбранные позиции и нажав кнопку OK, вы получаете список с попарными значениями числа прожитых дней. Если поставить галочку напротив только одной личности, программа выдаст список совместно прожитых дней выбранной личности со всеми остальными личностями, имеющемися в списке персональных данных.
Щелчок на позиции списка совместно прожитых дней  открывает экран, на котором можно
рассчитать "круглую" дату для выбранной пары.

При щелчке на пункте БИОРИТМЫ вы сначала попадаете на экран выбора личности из списка персональных данных. Щелчок на строке списка открывает экран
с графиком биоритмов выбранной личности, по которому можно перемещаться с помощью кнопок навигации, либо путём выбора нужной даты в календаре

Справа сверху экрана Главного меню (три вертикальные точки) находится выпадающее меню с пунктом "Настройки".
В Настройках вы можете задать количество точек, отображаемых на  графике биоритмов, а также выбрать режим сортировки списка персональных данных -
по имени или по дате рождения.При отказе от сортировки персональные данные будут отображаться в списке  в порядке их добавления

СПРАВОЧНАЯ СИСТЕМА
Для облегчения работы с программой на каждом экране имеется файл справки, доступ к которому открывается при нажатии на знак вопроса или на три вертикальные точки в верхней части экрана (на старых телефонах - через аппаратную кнопку "Меню")
