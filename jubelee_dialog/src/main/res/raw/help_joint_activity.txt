
СОВМЕСТНАЯ ДАТА
На экране отображаются персональные данные и количеcтво прожитых дней для пары выбранных личностей.

Также можно произвести расчёт даты наступления заданного количества совместно прожитых дней, для чего желаемое количество совместно прожитых дней нужно ввести в поле "Задайте количество совместно прожитых дней" и затем нажать кнопку "Рассчитать".

Замену личности можно сделать, нажав на кнопку с именем личности, которую требуется заменить, и выбрав требуемую личность во всплывающем окне

На панели действий, расположенной в верхней части экрана, при щелчке по значку лупы со стрелкой происходит
переход к окну выбора личностей для расчёта количества совместно прожитых дней.
После установки галочек напротив желаемых личностей и нажатия на кнопку ОК в нижней части окна происходит переход к экрану со списком количества совместно прожитых дней для каждой пары отобранных личностей.
Щелчок на позиции списка открывает экран, на котором можно рассчитать "круглую" дату для ранее выбранной пары.

Справа сверху экрана (три вертикальные точки) находится выпадающее меню с пунктами "Отправить" "Настройки"и "Справка".

Выбрав "Отправить" вы получаете возможность отправить скриншот экрана с помощью одной из программ, появившихся в диалоговом окне с заголовком "Отправить с помощью".

Выбрав "Настройки", вы можете задать количество точек, отображаемых на  графике биоритмов, а также выбрать режим сортировки списка персональных данных -
по имени или по дате рождения.

При отказе от сортировки персональные данные будут отображаться в списке  в порядке их добавления
